Predicting Life Expectancy

A User-Friendly Application on US State-Level Data

Design your own Data Driven Predictive Model

This application gives users the ability to define their own model and make predictions of US state-level life expectancy rates by setting desired predictive variable values

The Data

US state-level data is used to create predictive models of Life Expectancy (in years). Models can be built from the following predictive variables:

  1. Murder rate per 100,000 population
  2. Percent high-school graduates
  3. Mean number of frost days

All you have to do is

  1. Select which variables to include
  2. Choose the values of the selected variables
  3. Calculate your predictive results and visualize this in a reactive graph
